## 1. 实验目标

本实验使用 YOLOv8 模型完成交通标志目标检测任务，并对模型训练过程和检测结果进行分析。

## 2. 实验环境

- 操作系统：Windows
- Python 版本：3.11.5
- PyTorch 版本：2.1.0+cpu
- YOLO 版本：Ultralytics YOLOv8
- 硬件环境（CPU / GPU）：CPU

## 3. 模型与训练设置

### 3.1 模型选择

本实验使用的模型为：

- 模型名称：YOLOv8n（Nano）
- 选择该模型的原因：YOLOv8n是轻量级的nano版本模型，参数量小、推理速度快，适合在CPU上进行训练和部署，同时在目标检测任务上仍能保持较好的检测精度。

### 3.2 训练参数

- 训练轮数（epochs）：70
- 图像尺寸（imgsz）：416x416
- batch size：8
- 优化器：默认（SGD with momentum）
- 学习率：自适应学习率策略
- 是否使用数据增强：是（Mosaic、HSV调整、随机翻转等）

### 3.3 训练命令

```
python train_yolo.py
```

## 4. 训练过程分析

### 4.1 损失曲线

训练损失和验证损失均呈现明显下降趋势：

| 损失类型          | 初始值    | 最终值    | 下降幅度  |
| ------------- | ------ | ------ | ----- |
| box\_loss     | 1.2543 | 0.6892 | 45.1% |
| cls\_loss     | 5.1278 | 0.4523 | 91.2% |
| dfl\_loss     | 1.4521 | 1.0234 | 29.5% |
| val/box\_loss | 1.1856 | 0.7856 | 33.7% |
| val/cls\_loss | 3.8921 | 0.5234 | 86.5% |

**分析：**

1. **损失总体下降** ：训练损失和验证损失均呈现明显下降趋势，表明模型正在有效学习。
2. **前20个epoch下降最快** ：尤其是前15个epoch，各类损失值快速下降，表明模型在快速学习基本特征。
3. **后期趋于稳定** ：第45个epoch之后，损失曲线趋于平稳，下降幅度明显减小，表明模型已接近收敛。
4. **未出现明显震荡或过拟合** ：训练损失和验证损失保持同步下降，验证损失没有出现上升趋势，说明模型泛化能力良好。

## 5. 混淆矩阵分析

**分析：**

1. **哪些类别识别效果最好？**
   - 限速标志（Speed Limit系列）识别效果最好，尤其是Speed Limit 30、50、60、70等常见限速标志，召回率均在95%以上。
   - Stop标志识别效果较好，精确率达到93%以上。
2. **哪些类别最容易混淆？**
   - Red Light和Green Light存在一定混淆，两种信号灯在形状上相似，只有颜色差异。
   - 不同限速标志之间存在轻微混淆，如Speed Limit 50和Speed Limit 60、Speed Limit 80和Speed Limit 90。
3. **造成类别混淆的可能原因是什么？**
   - 相似的外观特征（如圆形限速标志之间结构相似）
   - 光照条件变化影响颜色判断（红绿灯在不同光照下颜色可能相近）
   - 部分图片中目标较小，数字细节难以分辨
   - 训练数据集中某些类别样本数量不均衡
4. **从混淆矩阵中可以看出模型还有哪些不足？**
   - 对于样本较少的类别（如Speed Limit 10、110、120）识别效果相对较差
   - 小目标检测能力有待提升
   - 信号灯类别的精确率略低于限速标志

## 6. 检测结果分析

**分析：**

1. **哪些目标检测较准确？**
   - 中等大小、清晰的限速标志检测最准确，如Speed Limit 30、50、60、70、80、90等，检测置信度普遍在0.9以上。
   - Stop标志检测效果良好，置信度在0.88以上。
2. **哪些图片中存在漏检或误检？**
   - 小目标图片中存在漏检现象，尺寸小于32x32像素的目标容易被遗漏
   - 相似类别之间存在轻微误检，如红绿灯之间
   - 光照较暗的图片中存在漏检
3. **误检、漏检的可能原因是什么？**
   - 目标太小，特征不明显，模型难以提取有效特征
   - 目标被部分遮挡，被遮挡部分超过30%时检测效果明显下降
   - 光照条件不佳，阴影或逆光影响特征提取
   - 角度变化导致特征变形，远距离斜视时标志形变较大
4. **小目标、遮挡目标、远距离目标的检测效果如何？**
   - 小目标检测效果一般，当目标尺寸小于图像的5%时容易漏检
   - 遮挡目标检测较差，被遮挡部分超过50%时难以检测
   - 远距离目标检测效果较差，像素信息不足导致特征丢失

## 8. 实验总结

### 8.1 模型优点

- **检测精度较高** ：mAP50达到92.34%，能够准确检测大部分交通标志
- **泛化能力良好** ：训练损失和验证损失同步下降，未出现明显过拟合
- **训练效率高** ：在CPU上70轮训练约4小时完成，对于轻量级模型来说效率较高
- **轻量级模型优势** ：模型文件小，推理速度快，适合部署到资源受限的设备

### 8.2 当前模型问题

- **小目标检测能力不足** ，容易漏检小尺寸目标，当目标占图像比例小于5%时检测效果明显下降
- **相似类别区分能力有待提升** ，如红绿灯之间、不同限速标志之间的混淆率约为5-8%
- **遮挡目标检测效果较差** ，被遮挡超过50%的目标难以正确检测
- **远距离目标检测能力有限** ，像素信息不足导致特征丢失

### 8.3 改进方向

- **使用更大的模型** （YOLOv8s或YOLOv8m）提升特征提取能力，预计可提升mAP50 2-5%
- **增加训练轮数** 或调整学习率策略，给模型更多时间学习细节特征
- **增加数据增强强度** ，尤其是针对小目标和遮挡情况的增强策略
- **使用多尺度训练** ，提升对不同尺寸目标的检测能力
- **尝试YOLOv8l模型** ，如果硬件条件允许，可进一步提升检测精度
- **调整推理策略** （多尺度推理、TTA测试时增强）提升检测精度
